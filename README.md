# my-portfolio
Personal portfolio website with projects, resume, and contact information.

## Deploy to GitHub Pages

This project is configured for GitHub Pages static export with Next.js.

1. Push to `main`.
2. In GitHub, open **Settings > Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Wait for the `Deploy to GitHub Pages` workflow to finish.

Your site URL will be:

`https://omesh-s.github.io/my-portfolio/`
